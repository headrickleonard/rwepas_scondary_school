# Tanzanite Academy Website

A premium marketing website for Tanzanite Academy, an elite secondary school in Dar es Salaam, Tanzania.

Built with Next.js 15, TypeScript, Tailwind CSS, shadcn/ui, Framer Motion, and Lenis.

## Getting Started

```bash
npm install
npm run dev
```

## Design Philosophy

This website embodies excellence, trust, and modern African education. Every detail has been crafted to feel luxurious, editorial, and human.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui + Radix UI
- Framer Motion
- Lenis (smooth scrolling)
- Lucide Icons
- Sonner (toasts)

## Features

- Elegant, spacious, premium UI
- Smooth scrolling with Lenis
- Subtle Framer Motion animations
- Fully responsive
- Beautiful African student imagery
- Interactive admissions form
- Animated statistics
